Sobre mi

Soy un profesional responsable, atento al detalle, con muchas ganas de seguir adquiriendo experiencia como desarrollador web, apasionado por la musica, la naturaleza y les invito a ver mis proyectos los cuales he realizado con todo lo aprendido en este mundo del desarrollador.

Mis proyectos

Proyecto CSS

Se implementa CSS utilizando archivos hojas de estilo externas al documento HTML y de base para homogeneizar el diseño del sitio, ingreso de assets, bordes, margen, padding y selectores de etiqueta o clase según sea necesario.

Proyecto Flexbox

Validación de conocimientos de posicionamiento en CSS, diferenciar los valores asociados a display (inline, block, inline-block), utilizar floats para posicionar texto e imágenes, y asi resolver el problema de colapso de floats utilizando un clearfix.

Proyecto Bootstrap

Implementación de sistema de grillas de Bootstrap, distribuyendo el contenido adecuadamente, para hacer una página web adaptable a distintos dispositivos, aplicando las clases de Bootstrap a través de los componentes card y formularios.
